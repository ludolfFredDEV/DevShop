# DevShop 🛒

Este é um projeto de e-commerce desenvolvido para praticar o desenvolvimento de aplicações web.

## 🚀 Tecnologias
Este projeto utiliza:
* **Node.js**
* **NPM** (Gerenciador de pacotes)
* **JavaScript**

## 📦 Como rodar o projeto localmente

1. **Clone o repositório:**
   ```bash
   git clone [https://github.com/seu-usuario/seu-repositorio.git](https://github.com/seu-usuario/seu-repositorio.git)


2. **Instale as dependências:** (Isso vai recriar a pasta node_modules que não subiu para o Git)
    ```bash
    npm install 

3. **Inicie a aplicação:** 
    ```bash
    npm start
